# Yang–Mills SU(3) v7 — Notebook Simulation

Simulaciones SU(2) → SU(3) construidas desde cero en Python:
- Acciones de Wilson
- Heatbath SU(2) embebido en SU(3)
- Staples corregidos
- Proyección unitaria SU(3)
- Glueball correlators
- Estimación preliminar de mass-gap

v7 es la primera versión que produce dinámica SU(3) estable en hardware común.

Código abierto para uso, análisis y extensión.
